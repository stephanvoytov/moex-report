# MOEX Report Generator

Скрипт автоматически получает данные индексов Московской биржи и курс USD/RUB с ЦБ РФ и создаёт Word-отчёт (`.docx`) с таблицами, дневной сводкой и сводной аналитикой.

## Поддерживаемые индексы

- IMOEX — Индекс Мосбиржи
- RTSI — Индекс РТС
- RGBITR — Индекс гособлигаций
- RUCBTRNS — Индекс корпоративных облигаций

---

# Возможности

Скрипт автоматически:

- загружает данные MOEX и курс USD/RUB с ЦБ РФ
- получает:
  - open, high, low, close
  - объём
  - yield, duration
- считает:
  - дневное изменение % к предыдущему торговому дню
  - итог периода
  - максимум / минимум за период
- создаёт Word-отчёт:
  - дневная сводка с ценами закрытия и курсом USD/RUB
  - подробные таблицы по каждому индексу
  - сводная таблица (close + change %, итог, макс, мин)
- авто-режим: отчёт за прошлую неделю в 1 клик
- авто-открытие файла после создания

# Запуск

## Способ 1 — двойным кликом (рекомендуется)

Просто дважды кликните на файл **`run.bat`** в папке проекта. Откроется окно с меню.

## Способ 2 — из терминала (командной строки)

Откройте **Командную строку** (cmd) или **PowerShell**:

1. Перейдите в папку проекта:
   ```bash
   cd C:\Users\stepa\PycharmProjects\parse
   ```
2. Запустите скрипт:
   ```bash
   .venv\Scripts\python main.py
   ```

## Меню при запуске

```
Выберите режим:
  1 — Отчёт за прошлую неделю (авто)
  2 — Указать даты вручную
```

**Режим 1** — автоматически определяет понедельник–пятницу прошлой недели и формирует отчёт. Ничего вводить не нужно.

**Режим 2** — позволяет указать произвольный диапазон дат (включительно).

Пример ручного ввода:

```
2026-05-11
2026-05-15
```

Даты включаются полностью. Изменение % для первого дня считается относительно предыдущего торгового дня (скрипт автоматически загружает данные за 14 дней до указанной даты).

# Структура отчёта

## Дневная сводка

Для каждого дня в диапазоне:

- дата (на русском: Понедельник 11 мая 2026)
- цены закрытия и изменения % для всех индексов
- курс USD/RUB (курс ЦБ)

## Подробные таблицы

Для каждого индекса:

- дата
- open, high, low, close
- изменение %
- объём
- yield, duration (для облигаций)

## Сводная таблица

Содержит:

- ежедневные изменения по всем индексам (close + change %)
- итог периода
- максимум / минимум за период

# Формат объёмов

Скрипт автоматически конвертирует:

- IMOEX → млрд ₽
- RTSI → млн $
- RGBITR → млрд ₽
- RUCBTRNS → млрд ₽

---

# Курс USD/RUB

Курс доллара загружается с ЦБ РФ (`XML_dynamic.asp`) за весь период.
Если на конкретную дату курс отсутствует, берётся предыдущий доступный.

# Примечания

- Если на какую-то дату нет торгов (праздник / выходной), этот день просто пропускается
- Первый запуск может быть долгим из-за загрузки зависимостей
- При ошибках сетевого подключения скрипт автоматически повторяет запрос до 5 раз
